# 코드 스타일
- 들여쓰기: 스페이스 2칸
- 세미콜론 사용하지 않음
- 작은 따옴표('')  사용
- 변수명: CamelCase 사용
- 함수명: 동사로 시작 (예:getUserData, handleClick)
- 함수에는 간단한 JSDoc 주석 추가
- console.log 대신 적절한 로깅 라이브러리 사용
- 한글 주석 사용

# 커뮤니케이션
- 코드 변경 시 변경 이유를 간단히 설명
- 에러 발생 시 원인과 해결 방법을 함께 제시

# 자주 쓰는 명령어
- 빌드: npm run build
- 테스트: npm run test
- 린트: npm run lint
